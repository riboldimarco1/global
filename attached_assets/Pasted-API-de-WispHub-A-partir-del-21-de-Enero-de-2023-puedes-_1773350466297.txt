¿Cómo generar el API Key de WispHub?
El API de WispHub implementa la autorización por claves de API. Esto le permite generar claves API a su personal desde la Plataforma de WispHub en la Lista de Personal con el boton "Generar Mi APIKey" (boton color azul del punto número 2).

Usuario Administrador Para Generar El APIKey
Para generar el APIKey es importante crearlo con el usuario ADMIN, el cual le es proporcionado al crear su cuenta; . Se recomienda esto debido a temas de seguridad a su cuenta, adicional a que este cuenta con todos los permisos.


Habiendo creado ingresado con el usuario recomendando se puede proseguir a generar el APIKey

Guarde su Clave de API
  
Por favor, asegurese de guardar en un lugar seguro su clave del API. Por seguridad, estas claves se mostrarán unicamente al crearlas y no podrá volver a consultarla.


Debe tener en cuenta que solo es posible generar una Clave de API a un personal de toda su empresa. Si ya existe un personal con una clave de API, WispHub le notificará cual es el personal que tiene asignada la clave.



¿Cómo eliminar el API Key de WispHub?
Para eliminar claves API a su personal desde la Plataforma de WispHub en la Lista de Personal con el boton "Eliminar Mi APIKey" (boton color rojo).

Usuario Administrador Para Eliminar El APIKey
Para eliminar el APIKey es importante eliminarlo con el usuario ADMIN.Se recomienda esto debido a temas de seguridad a su cuenta, adicional a que este cuenta con todos los permisos.
Eliminar ApiKey Wisphub
Al dar click "Eliminar mi ApiKey" la página nos mostrara un detalle de advertencia indicando que se eliminará la API Key.

Detalles de advertencia Eliminar ApiKey 

Hacemos clic en el botón “Sí, estoy seguro”.

Aparece una ventana emergente de confirmación para confirmar la eliminación de la API Key.

Confirmacion Eliminar ApiKey

Escribimos la palabra solicitada eliminar_apikey y hacemos clic en “Eliminar APIKey”.

El sistema nos regresa a la lista de staff, mostrando un mensaje de confirmación indicando que la API Key fue eliminada correctamente.

Mensaje final Eliminar ApiKey
¿Cómo usar el ambiente de pruebas (sandbox) del API de WispHub?
Entenedemos que durante el proceso de la integración del API deseas no afectar de ninguna los datos de tu empresa. Es por eso que WispHub pone a tu disposición el ambiente de pruebas.

Para utilizarlo debes crear una nueva empresa en el siguiente enlace y generar un Api Key como se muestra en el punto anterior. De esta forma puedes utilizar esta nueva empresa para experimentar con el API.

 

¿Cómo pedir soporte para el API de WispHub?
Si tienes alguna duda o inconveniente con el API de WispHub comunicate vía correo electrónico a soporte@wisphub.net

Es importante que sigas estrictamente el formato que se menciona a continuación para que puedas ser atendido por el equipo de soporte.
